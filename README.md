## Description

A pipeline system for querying, transforming and loading data.

## Installation

```bash
$ npm install
```

To run the application SQLLite is required:
`https://www.servermania.com/kb/articles/install-sqlite`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
